# Video Upload

## How to run the Front-end?
```
cd client
```
```
npm install
```
```
npm run dev
```
## How to run the Back-end?
```
cd server
```
```
npm install
```
```
node or nodemon server.js
```

## Used technologies:

- Frontend: React Js -> Axios
- Backend: node.js, express.js, multer, morgan, fs
- Database: MongoDB
- Git: GitHub
